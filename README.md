# Alert Card

visual plugin to display alerts

## Install

```shell
ipm install inkdrop-alert
```

## Usage

    ```
    [!warning]
    This is my alert
    ```

It will be rendered as:

![](https://github.com/jpastorm/inkdrop-alert/raw/master/example.png)


## Commands
- [!warning]
- [!note]
- [!tip]
- [!caution]
- [!information]
